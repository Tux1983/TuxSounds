# TuxSounds

An independent archive project dedicated to preserving the historical system sounds of the GNU operating system and the GNOME desktop environment.

This repository serves as a central collection for legacy audio events that have shaped the free software desktop experience. The goal is to document and provide open access to these system sounds for historical preservation.

## Archive Content
The collection includes original system audio files sourced from historical GNOME releases. All files are provided in their original format to maintain historical integrity.
